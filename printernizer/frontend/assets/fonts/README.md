# Bundled fonts

Used by the browser-side model generator (text / nameplate / keychain template)
to convert glyphs into extrudable outlines via opentype.js.

## LiberationSans-Regular.ttf

- **Family:** Liberation Sans (metric-compatible with Arial)
- **License:** SIL Open Font License 1.1 (OFL)
- **Source:** https://github.com/liberationfonts/liberation-fonts

Self-hosted so the generator works offline (e.g. on a Raspberry Pi / Home
Assistant host without outbound CDN access).
