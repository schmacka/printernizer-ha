# Printernizer Home Assistant Addon

This addon runs [Printernizer](https://github.com/schmacka/printernizer) in Home Assistant.

## Installation

The addon image is hosted on GitHub Container Registry (ghcr.io). Current version: 0.0.5

### Method 1: Using the repository
1. Add this repository to your Home Assistant add-on store
2. Install the "Printernizer" add-on
3. Start the add-on
4. Access Printernizer via the sidebar

### Method 2: Manual installation
1. Copy this folder to your Home Assistant `addons` directory
2. Build and install the addon from the Home Assistant UI
3. Access Printernizer via the sidebar

## Configuration

No configuration required for basic usage.
