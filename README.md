# Printernizer Home Assistant Add-ons

[![Home Assistant Add-on](https://img.shields.io/badge/Home%20Assistant-Add--on-blue.svg)](https://www.home-assistant.io/addons/)

This repository contains Home Assistant add-ons for Printernizer.

## Installation

1. Open Home Assistant
2. Go to **Settings** > **Add-ons** > **Add-on Store**
3. Click the menu (three dots) in the top right corner
4. Select **Repositories**
5. Add this repository URL: `https://github.com/schmacka/printernizer-ha`
6. Click **Add** and then **Close**
7. The Printernizer add-on should now appear in the add-on store

## Available Add-ons

### [Printernizer](./printernizer)

Professional 3D Printer Management for Bambu Lab A1 and Prusa Core One.
